# Anual-Sales-Dashboard
Vrinda Store Annual Sales Dashboard is an Excel-based analytics project analyzing 2022 retail sales. Using pivot tables, charts, and slicers, it converts raw data into an interactive dashboard that answers key business questions on sales trends, customer demographics, order status, states, channels, and categories.

#  Vrinda Store Annual Sales Dashboard (Excel)

##  Project Overview
This project showcases an **interactive Excel dashboard** built to analyze the **annual sales performance of Vrinda Store (2022)**.  
The dashboard was designed to answer **real-world business questions** related to sales trends, customer behavior, order performance, geography, and sales channels using Microsoft Excel.

The goal of this project is to demonstrate **end-to-end Excel-based data analysis**, from raw data to actionable insights, suitable for a **Data Analyst portfolio**.

---

##  Business Questions Answered
The dashboard answers the following key business questions:

- Compare **sales and number of orders** using a single chart  
- Which **month recorded the highest sales and orders**?  
- Who purchased more in 2022 â€” **men or women**?  
- What are the **different order statuses** and their distribution?  
- Which are the **top contributing states** to overall sales?  
- What is the **relationship between age group and gender** based on number of orders?  
- Which **sales channel contributes the maximum sales**?  
- Which is the **highest-selling product category**?

---

##  Dataset Description
- The dataset used is a **sample retail sales dataset** for the year 2022.
- It contains transactional-level data, including:
  - Order ID  
  - Order Date & Month  
  - Customer Gender  
  - Age Group  
  - Product Category  
  - Sales Amount  
  - Shipping State  
  - Order Status  
  - Sales Channel  

>  The dataset and dashboard are maintained within a single Excel workbook.
